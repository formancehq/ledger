## Upgrading Speakeasy

To upgrade speakeasy to a new version (e.g. `v1.752.0`):

1. **Get checksums**: `gh release download v<VERSION> --repo speakeasy-api/speakeasy --pattern 'checksums.txt' --output -`
2. **Update `flake.nix`**:
   - Set `speakeasyVersion` to the new version (without `v` prefix)
   - Update `speakeasyHashes` with SHA256 values from checksums.txt, mapping:
     - `x86_64-linux` ← `speakeasy_linux_amd64.zip`
     - `aarch64-linux` ← `speakeasy_linux_arm64.zip`
     - `x86_64-darwin` ← `speakeasy_darwin_amd64.zip`
     - `aarch64-darwin` ← `speakeasy_darwin_arm64.zip`
3. **Update `pkg/client/.speakeasy/workflow.yaml`**: set `speakeasyVersion` to new version (with `v` prefix)
4. Lock files (`workflow.lock`, `gen.lock`) are regenerated by `speakeasy run` (called during `just pre-commit` if `SPEAKEASY_API_KEY` is set)
5. Run `just pre-commit` to validate
